# ملفات اللوقو وأيقونة التبويب

تم تجهيز اللوقو الحقيقي في ثلاث نسخ:

- `logo-full.png` النسخة الكاملة من اللوقو، وتظهر في الصفحة الرئيسية.
- `logo-mark.png` نسخة خفيفة مربعة للهيدر.
- `favicon.png` نسخة صغيرة لأيقونة التبويب فوق في المتصفح.

لو حبيت تغير اللوقو لاحقًا، استبدل الملفات بنفس الأسماء عشان صفحات HTML ما تحتاج أي تعديل.
